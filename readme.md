## 概要

青を基調としたエンジニア向けのポートフォリオを簡単に構築できます。

Pug を使用しており、コンポーネントを使いまわしつつ、変数を変更することで微調整できるようデザインされています。

Node.js のインストールが必要です。

ブラウザでの動作確認が容易になるよう、Browser-Sync を使用しています。

## 開始方法

```bash
$ git clone https://github.com/jaspis1997/portfolio-site-skeleton.git
$ cd portfolio-site-skeleton
$ npm install
$ npm start
```
